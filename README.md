# ابزار QR کد — ساخت و اسکن آنلاین

سایت استاتیک ساخت و اسکن QR کد، فارسی و راست‌چین، **کاملاً سمت کلاینت** (هیچ سروری لازم ندارد و هیچ داده‌ای جایی ارسال نمی‌شود).

## امکانات

### ساخت QR کد
- **۷ نوع محتوا**: لینک، متن، ایمیل (گیرنده/موضوع/متن)، تماس، پیامک، وای‌فای (SSID/رمز/نوع رمزنگاری/شبکه مخفی) و کارت ویزیت (vCard کامل)
- **شخصی‌سازی**: رنگ نقاط + گرادیان دو رنگ، رنگ/شفافیت پس‌زمینه، شکل نقاط (۶ حالت)، شکل گوشه‌ها، اندازه (۲۰۰ تا ۱۲۰۰ پیکسل)، حاشیه، سطح تصحیح خطا، لوگوی وسط کد و قاب (ساده/متن‌دار) با رنگ و متن دلخواه
- **خروجی**: PNG (قاب هم روی خروجی اعمال می‌شود) و SVG + کپی محتوا
- پشتیبانی کامل از متن فارسی (انکد UTF-8)

### اسکن QR کد
- اسکن زنده با دوربین (نیازمند HTTPS) و اسکن از روی تصویر (آپلود یا درگ‌ودراپ)
- تشخیص خودکار نوع محتوا و اکشن‌های مرتبط: باز کردن لینک، تماس، ارسال ایمیل/پیامک، کپی رمز وای‌فای، ذخیرهٔ مخاطب به‌صورت فایل `vcf`

## اجرا

فایل‌ها استاتیک‌اند؛ کافی است روی هر هاست استاتیکی قرار بگیرند (GitHub Pages، Netlify، …) یا لوکال سرو شوند:

```bash
python3 -m http.server 8000
# سپس: http://localhost:8000
```

> برای کارکردن دوربین در اسکنر، صفحه باید روی HTTPS (یا localhost) باز شود.

## ساختار

| فایل | توضیح |
|---|---|
| `index.html` | کل صفحه (تب ساخت + تب اسکن) |
| `css/qr-styles.css` | استایل کامل (RTL، ریسپانسیو) |
| `js/qr-tool.js` | منطق ساخت/شخصی‌سازی/دانلود/اسکن/تحلیل نتیجه |
| `js/vendor/` | کتابخانه‌های لوکال (بدون CDN): [qr-code-styling](https://github.com/kozakdenys/qr-code-styling) ساخت QR، [html5-qrcode](https://github.com/mebjas/html5-qrcode) اسکن دوربین، [jsQR](https://github.com/cozmo/jsQR) اسکن تصویر |

فونت (وزیرمتن) و آیکون‌ها (Font Awesome) از CDN لود می‌شوند و درصورت قطع بودن، سایت با فونت سیستم همچنان کار می‌کند.
